
Userbot для участия в розыгрышах в @mrkt (только по подписке)

1. Установи зависимости:
   pip install -r requirements.txt

2. В первый запуск:
   python userbot.py
   - Введи номер Telegram и код подтверждения

3. Бот будет автоматически:
   - Проверять розыгрыши каждые 5 минут
   - Подписываться на каналы
   - Выключать уведомления в новых каналах
   - Нажимать "Проверить"
   - Кидать лог в @sacoectasy
   - Сохранять log.txt локально
